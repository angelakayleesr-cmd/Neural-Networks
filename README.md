 Deskripsi Proyek
  Proyek ini merupakan implementasi kecerdasan buatan menggunakan arsitektur Neural Networks untuk menyelesaikan empat permasalahan berbeda menggunakan dataset publik berskala internasional. Proyek ini mencakup alur kerja machine learning secara menyeluruh, mulai dari pembersihan data (data cleaning), rekayasa fitur (feature engineering), pembangunan model, hingga evaluasi performa dalam konteks Sistem Informasi.

Dataset yang Digunakan
Proyek ini mengintegrasikan empat dataset utama dengan tugas prediksi yang bervariasi:

1. Online Retail II (UCI)          : Memprediksi loyalitas pelanggan berdasarkan perilaku belanja menggunakan klasifikasi biner.
2. Heart Failure Clinical Records  : Prediksi risiko kematian pasien gagal jantung berdasarkan indikator klinis (Klasifikasi).
3. Air Quality Dataset (Beijing)   : Prediksi konsentrasi PM2.5 menggunakan data *time-series* sensor udara (Regresi).
4. Hotel Booking Demand            : Memprediksi kemungkinan pembatalan reservasi oleh tamu hotel (Klasifikasi).
   

Metodologi Penelitian

Alur pengerjaan proyek mengikuti tahapan berikut:
1. Data Understanding: Eksplorasi struktur data dan identifikasi korelasi antar variabel.
2. Data Preparation  : 
    - Penanganan missing values melalui interpolasi dan imputasi.
    - Normalisasi data menggunakan StandardScaler.
    - Encoding variabel kategorikal.
3. Modelling         : Membangun arsitektur *Multi-Layer Perceptron* (MLP) menggunakan framework **TensorFlow/Keras**.
4. Evaluasi          : Pengujian model menggunakan skema *split data* 70:30 dengan metrik Akurasi, Loss, dan MAE.


Teknologi yang Digunakan
- Bahasa Pemrograman        : Python 
- Deep Learning Framework   : TensorFlow & Keras
- Analisis Data             : Pandas, NumPy, Scikit-Learn
- Visualisasi               : Matplotlib, Seaborn
- Platform                  : Google Colab
